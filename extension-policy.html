Privacy Policy

Effective Date: 21/11/2025

Introduction

Thank you for using the internal Chrome extension (the “Extension”) developed by Huan Luo. I respect your privacy. This privacy policy explains how the Extension handles user data, usage, and privacy protection.

About the Extension

The Extension is used exclusively in an internal environment and is intended to intercept specific requests (for example: 127.0.0.1:50018) initiated by the OA system. The Extension does not collect, transmit, store or share any personal data, usage information or browsing behavior.

Data Collection

  • The Extension does not collect personally identifiable information (such as names, email addresses, user IDs).
  • The Extension does not record or store browsing history, form input, URLs visited, or device identifiers.
  • The Extension does not send any data to external servers or third-party services.
  • The Extension does not use persistent storage (e.g., Chrome Storage API) for behavior logs.

Data Use and Sharing

Since there is no data collection, there is no usage, sharing, selling, or disclosure of user data, behavior logs or device information.

Security

Although the Extension does not handle user data, I maintain reasonable technical and organizational measures to ensure the integrity and security of the Extension, restrict permissions to the minimum required, conduct internal code reviews and testing before deployment.

Updates and Changes

I reserve the right to update this privacy policy at any time. Should the functionality of the Extension change in a way that involves data collection or processing of personal information, a revised policy will be published and users notified. Please review this policy periodically for updates.
